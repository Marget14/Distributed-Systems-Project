StreetFoodGo
===

---

Clone the repository:

```shell
git clone https://github.com/Marget14/Distributed-Systems-Project
```

---

Run the example:

```shell
cd StreetFoodGo
./mvnw spring-boot:run  # MacOS / Linux
./mvnw.cmd spring-boot:run  # Windows
```

---

Open in browser: [localhost:8080](http://localhost:8080)
